# 🚀 InsightPredict – Intelligent ML Data Analyzer Dashboard

InsightPredict is a GUI-based Machine Learning Data Analyzer built in Python that enables users to perform complete data analysis and machine learning workflows through an interactive dashboard.

The system integrates preprocessing, visualization, supervised and unsupervised learning, model comparison, and model interpretability into one streamlined application.

---

## 📌 Problem Statement

Performing end-to-end machine learning tasks often requires multiple scripts and strong programming expertise. Many users struggle with data preprocessing, algorithm selection, performance comparison, and understanding model behavior.

InsightPredict was developed to simplify and automate the entire ML workflow in a structured and user-friendly environment.

---

## 🎯 Objective

- Automate data preprocessing and model training  
- Enable easy comparison of multiple ML algorithms  
- Provide interactive visualizations  
- Improve model transparency using explainable AI  
- Make machine learning workflows more accessible and efficient

## ⚙️ Core Features

### 📂 Dataset Handling

---

### 🧹 Data Preprocessing

- Automatic problem type detection
